RAJVI PACKAGING — WEBSITE FILES
================================

HOW TO USE:
1. Replace each placeholder file below with your actual image/video
2. Keep the exact same filename and folder location
3. Open index.html with VS Code Live Server to test

FILES TO REPLACE WITH YOUR OWN:
---------------------------------
logo.png                            ← your logo
about-image.jpg                     ← about section photo
images/products/foam.jpg
images/products/air-bubble.jpg
images/products/epe-pouches.jpg
images/products/pp-packaging.jpg
images/products/gaskets.jpg
images/products/adhesive-lamination.jpg
images/products/sun-dry-board.jpg
images/products/tapes.jpg
videos/hero.mp4                     ← hero background video

IMPORTANT — FILE NAMING RULES:
---------------------------------
- All lowercase  (foam.jpg  NOT  Foam.JPG)
- No spaces      (air-bubble.jpg  NOT  air bubble.jpg)
- Correct format (use .jpg for photos, .mp4 for video)

TO HOST ON NETLIFY:
---------------------------------
1. Go to netlify.com
2. Drag the entire rajvi-packaging folder onto the deploy area
3. Done — your site is live
